# 📊 HR Analytics Dashboard – Power BI

## Overview
This project is an interactive **HR Analytics Dashboard** built using **Power BI** to analyze employee data and identify workforce trends. The dashboard helps organizations understand employee attrition, demographics, job satisfaction, salary insights, and department-wise performance.

## Dashboard Preview
![HR Analytics Dashboard](https://github.com/Lavakumarmarisarla/HR_Analytics_Dashboard/blob/main/HR_Data_Analytics_Dashboard_Screenshot.png)

## Key Metrics
- Total Employees: **1470**
- Attrition Count: **237**
- Attrition Rate: **16.12%**
- Active Employees: **1233**
- Average Employee Age: **37**
- Average Monthly Pay: **6503**

## Features

### Employee Analysis
- Employee distribution by age group and gender
- Active employee count
- Average employee age

### Attrition Analysis
- Attrition by age
- Department-wise attrition
- Attrition rate by job role
- Attrition by education field

### Job Satisfaction Analysis
- Job satisfaction rating by employee role

### Interactive Filters (Slicers)
Users can dynamically filter data based on:
- Gender
- Marital Status
- Business Travel
- Education Level

## Tools & Technologies Used
- **Power BI Desktop**
- **Power Query**
- **DAX**
- **Data Visualization**
- **HR Dataset**

## Insights Generated
- Highest attrition observed in Sales-related roles
- Majority of employees belong to the age group **25–44**
- R&D department shows a larger employee distribution
- Job satisfaction varies across different job roles
